Welcome to our app!

To get started, you have 2 options:

1. install customtkinter, CTkToolTip, requests, uszipcode, and tkintermapview using pip
 a. ex, go to command line (windows key + cmd + enter), then type: 'pip install customtkinter'
 b. repeat this for every non-standard package

-- OR --

2. install pipenv
 a. 'pip install pipenv'
 b. navigate to the current directory in the command line
 c. run 'pipenv sync'
 d. it should work!
